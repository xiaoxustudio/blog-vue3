技术栈：`vue3（全家桶）、elementUI-plus、tailwindcss、axios、wangeditor/editor（富文本编辑器）`

## 使用

本项目使用`yarn`包管理工具  

如果您没有安装yarn包管理，请执行以下命令：

```sh
npm i -g yarn # 安装yarn包管理工具
```

安装依赖
```sh
yarn # 安装依赖
```


## 部署

```sh
yarn dev # 开发
```

```sh
yarn build # 打包
```


## php配置

在php.ini下开启扩展openssl
```ini
extension=php_openssl.dll
```
